# litianc blog
## 访问入口
1. litianc.cn
2. litianc.eth.limo

## 添加文章
在 /_posts目录中添加文章md格式，在 /images/posts中新建文件夹，放置依赖图片等资源


## 依赖
1. [all-relative](https://www.npmjs.com/package/all-relative)

## 编译
```
jekyll build
cd _site && all-relative
```
